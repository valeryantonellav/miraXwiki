===== Component Overview =====

=== Base Maps ===
Provide map backgrounds such as satellite imagery, terrain and reference maps.

→ [[base_maps|Explore available basemaps]]

=== Networks ===
Access environmental observation networks, stations and time series.

→ [[networks|Explore available networks]]

=== Layers ===
Visualize project data through raster, vector and thematic layers.

→ [[layers|Explore layers]]

=== User Layers ===
Create and manage your own spatial data.

→ [[user_layers|Explore user layers]]

=== Models ===
Run simulations and analysis tools.

→ [[models|Explore models]]

=== Toolbar ===
Access measurement, drawing and analysis tools.

→ [[tools|Explore tools]]
