====== miraX Components ======

miraX is organized around several core components that support data visualization, analysis, modeling, and user interaction.

This section provides an overview of the main components available in the platform. Click on any component below to access more detailed documentation, including available datasets, tools, configuration options and usage examples.

===== Components =====

  * [[basemaps|Base Maps]]  
Background cartographic layers used as spatial context for analysis and visualization.

  * [[networks|Networks]]  
Public and proprietary monitoring networks providing real-time and historical data.

  * [[layers|Layers]]  
Geospatial data layers available in the project, including raster and vector datasets.

  * [[user_layers|User Layers]]  
User-created or imported layers for custom analysis and collaboration.

  * [[models|Models]]  
Simulation and analysis tools integrated into miraX.

  * [[toolbar|Toolbar]]  
Interactive tools for exploration, measurement, drawing and analysis.
